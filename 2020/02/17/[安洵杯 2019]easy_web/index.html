命令执行绕过

Linux下的命令执行有很多绕过方法，空格使用$IFS$n（n为任意数字）代替，过滤关键字可以用变量拼接绕过，例如a=l;${a}s;即为ls，还可以在命令中穿插\，不会影响命令执行，如c\at /fl\ag

正则匹配与转义

PHP在进行正则匹配时需要经过两层转义才能获取到真实的字符，第一层为PHP自身的转义，第二层为正则引擎的转义，因此\\\\四个反斜杠才能匹配到一个反斜杠\，而该题中\\|\\\\这一段经过两重转义之后变成了匹配|\，反而不能匹配单个的反斜杠了，所以光看还是不够，有的地方还是得自己本地跑一下才知道效果
